# homework_bot

Бот для проверки статуса домашнего задания в Yandex Practicum. Использует API Практикума.

Установка:
1. Склонируйте проект с гитхаба:

```bash
git clone https://github.com/Gasuaska/homework_bot.git
```

3. Запустите программу:
```bash
python manage.py runserver
```

Для остановки программы нажмите в терминале Ctrl + C.
